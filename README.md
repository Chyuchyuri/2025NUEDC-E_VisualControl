# Servo-and-Visual-Control-Based-on-STM32F103C8T6-and-OpenMV---Basic-Part-of-2025-NUEDC-Problem-E
本项目为 2025 年全国大学生电子设计竞赛 E 题基础实现方案，包含基于 STM32F103C8T6 的舵机控制程序以及 OpenMV 视觉模块的色块识别与追踪代码。This project is a basic solution for Problem E of the 2025 National Undergraduate Electronic Design Competition, including STM32F103C8T6-based servo control code and OpenMV color block recognition &amp; tracking code.
